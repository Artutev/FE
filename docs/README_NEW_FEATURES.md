## 🌙 Темная тема

### Как использовать:
1. **Откройте сайт** → http://127.0.0.1:8000
2. **Нажмите на кнопку** 🌙 в правом верхнем углу заголовка
3. **Готово!** Тема переключилась и сохранится автоматически

### Что произойдёт:
- 🎨 Интерфейс переключится на тёмный режим
- 💾 Ваш выбор сохранится в браузере
- 🔄 При следующем посещении тема восстановится

## 🔍 Поиск мероприятий

### Как использовать:
1. **Перейдите** на страницу "Мероприятия"
2. **Введите** название, описание или место в поле поиска
3. **Нажмите Enter** или кнопку 🔍
4. **Смотрите результаты** — найдено X мероприятий

### Примеры:
```
Введите "футбол" → Найдёт все события про футбол
Введите "москва" → Найдёт события в Москве
Введите "лекция" → Найдёт все лекции
```

### Очистить поиск:
- Нажмите кнопку **✕** рядом с поиском
- Или перейдите на страницу "Мероприятия" заново

## ❓ Часто задаваемые вопросы

**Q: Где сохраняется выбор темы?**  
A: В браузере (`localStorage`). При удалении данных браузера использует системные настройки.

**Q: Работает ли поиск мероприятий без учёта регистра?**  
A: Да! "Футбол" = "ФУТБОЛ" = "футбол"

**Q: Могу ли я видеть свои предпочтения в админке?**  
A: Да! Откройте Django Admin → Users → Профиль → поле "Предпочитаемая тема"

**Q: Как добавить свой цвет в темную тему?**  
A: Отредактируй CSS переменные в `static/css/base.css` в блоке `html.dark-mode`
