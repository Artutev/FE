# Документация: Темная тема в FastEvent

## Описание

В проект FastEvent добавлена полная поддержка тёмной темы. Пользователи могут переключаться между светлой и тёмной темой, а выбор сохраняется в браузере.

## Компоненты

### 1. **CSS переменные** (`static/css/base.css`)
- Добавлены CSS переменные для светлой и тёмной темы
- Тёмная тема активируется при добавлении класса `dark-mode` на элемент `<html>`
- Переменные используются: цвет фона, текста, границ, теней и т.д.

### 2. **JavaScript** (`static/js/theme.js`)
- Автоматический выбор темы на основе:
  1. Сохранённого выбора пользователя в `localStorage`
  2. Системных настроек (prefers-color-scheme)
  3. По умолчанию — светлая тема

### 3. **Кнопка переключения** (в `Users/templates/base.html`)
- Кнопка в заголовке для быстрого переключения темы
- Иконка меняется: 🌙 (ночь) → ☀️ (день)

### 4. **Модель User** (`Users/models.py`)
- Новое поле `theme_preference` (опциональное, для будущего расширения)
- Выбор: 'light', 'dark', 'auto'

## Установка и миграция

### Шаг 1: Применить миграцию
```bash
python manage.py migrate Users
```

Это создаст новое поле `theme_preference` в таблице User.

### Шаг 2: Проверить в админке
- Откройте Django Admin (`/admin/`)
- Сразу видна новая кнопка переключения темы в заголовке

## Как это работает

### Для посетителей без аккаунта:
1. Выбор темы сохраняется в `localStorage` браузера
2. При повторном посещении тема восстанавливается автоматически
3. При удалении данных браузера используется системная настройка

### Для авторизованных пользователей (в будущем):
- Можно сохранять предпочтение на сервере в поле `theme_preference`
- Синхронизировано на всех устройствах

| Элемент | Светлая | Тёмная |
|---------|---------|--------|
| Фон | #f3f6fb | #0f172a |
| Поверхность | #ffffff | #1e293b |
| Текст | #1f2937 | #f1f5f9 |
| Приглушённый текст | #6b7280 | #cbd5e1 |
| Граница | #dbe4f0 | #475569 |
| Акцент | #2563eb | #3b82f6 |

## Как добавить тёмную тему к другим компонентам

Если добавляете новые CSS классы с hardcoded цветами:

```css
/* Светлая тема */
.my-element {
    background: #ffffff;
    color: #1f2937;
}

/* Добавьте тёмную тему */
html.dark-mode .my-element {
    background: #1e293b;
    color: #f1f5f9;
}
```

Или используйте CSS переменные:
```css
.my-element {
    background: var(--surface);
    color: var(--text);
}
```

## Тестирование

1. Откройте сайт и нажмите на кнопку 🌙 в заголовке
2. Тема должна переключиться с плавным переходом
3. Перезагрузите страницу — тема должна сохраниться
4. Откройте DevTools (F12) → Application → Local Storage и найдите `theme_preference`
